# Müşteri Bilgi Sistemi
Bu proje, kullanıcıdan müşteri bilgilerini alıp ekrana yazdıran basit bir Python uygulamasıdır.

## Özellikler
- Ad, soyad, cinsiyet gibi bilgileri alır
- TC kimlik numarası ve adres bilgisi alır
- Girilen bilgileri düzenli şekilde ekrana yazdırır

## Kullanılan Teknoloji
- Python
